# Daily Stock Pick

One algorithmic stock pick per weekday — or "no stock worth buying today."

**Live site: https://sritham.github.io/daily-stock-pick/**

Every pick is tracked automatically against real market prices: filled at the
next open (or voided if it gaps above the buy zone), then closed by its stop,
its target, or a 20-day time stop. The track record on the site is exactly
what the algorithm said, with zero manual reporting.

Raw data: [data.json](data.json). Not financial advice.
